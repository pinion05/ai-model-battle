## 主な調査結果

### 価格ショック
- GLM 5.2: $1.40/M input (オープンウェイト、無料ダウンロード)
- GPT 5.6 Sol: ~$2.00/M input
- Fable 5: $5.00/M input, $25/M output (GLMより**11倍**高価)
- あるタスク: GLM $2, Fable $14, GPT 5.5 $37

### 「単一の最良モデルなし」
- 18本の動画 (39%) が「勝者はタスクによって異なる」と結論付けた
- ベンチマークスコア92と88の違いは、実際の使用では知覚できない
- 最も一般的な結論: 「仕事に適したモデルを選ぶ」

### オープンソースの逆襲
- GLM 5.2 (無料) は、プレミアムモデルと同等かそれ以上の性能を頻繁に示した
- Minecraftクローン、3D太陽系、ウェブデザイン — GLMが優勢
- 「おかしいと思われるかもしれないが、Claude Fable 5が作ったものより、こっちの方が好きだ」

## 方法論

1. **動画収集**: 49本の比較動画をYouTubeで検索
2. **トランスクリプト抽出**: yt-dlp + bgutil PO Token + tor SOCKSプロキシ (49本中46本成功)
3. **AI分析**: Xiaomi MiMo-V2.5-Pro API、1動画 = 1シリアル分析 (完全なトランスクリプト)
4. **翻訳**: OpenRouter経由のGoogle Gemini 2.5 Flash
5. **サイト生成**: インラインJSON + ダークテーマカードコンポーネント

### トランスクリプト抽出パイプライン

YouTubeの4層アンチボットシステムをバイパス:

```
yt-dlp (メタデータ、直接IP)
    -> キャプションURL + 署名
tor SOCKS5プロキシ (キャプションダウンロード、異なるIP)
    -> json3キャプションデータ
bgutil-ytdlp-pot-provider (PO Token生成)
    -> ボット検出バイパス
```
